Digit Recognition Network Set-up for Emergent

First Things

Emergent has already been installed on the CS department Linux hosts, just type emergent to run it. You may also install Emergent on your own machine, binaries are available for Windows, Mac and Linux. See the project home page Emergent for installation instructions or more information about Emergent itself.

Running Emergent

  1. Copy hwk2-train.dat that you created to emergent/hwk2-train.dat. Note that the original emergent/hwk2-train.dat is just an example and contains a handful of entries from the test pattern file, i.e. don't use it.
  2. Type emergent at the command line to start Emergent.
  3. Click the File menu and select Open Project..." and select the file your_hw2_folder/emergent/HW2_Backprop.proj in the file selector window that pops up, and click Ok.
  4. You should see the following window: Main window
  5. Expand the InputData list in the left pane, select TrainingInputData, click the Load Any Data botton in the middle pane, and select your hwk2-train.dat file: Load Training Data
  6. Return to the Control Panel tab, click Train_init then Train_run
  7. To run the test data select programs>>BpAll_Test>>BpEpochTest from the left pane and click run>
